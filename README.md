# Experimenting clean architecture with go language

## Requirements

 - Golang version 1.7+
 - Glide; click https://github.com/Masterminds/glide
 - other needed modules

## Installing

 - Install Glide https://github.com/Masterminds/glide
 - Install Dependecies
    ```console
    $ glide install
    ```

## Run
 - Run the program
   ```console
   $ go run main.go
   ```
